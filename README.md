# 📈 Stock Value Prediction System using Machine Learning

A Full Stack Machine Learning based Stock Market Prediction System that predicts stock prices using historical stock market data and displays the results through an interactive web application.

---

# 🚀 Project Overview

This project combines:

- 📊 Machine Learning for stock prediction
- ⚙️ Spring Boot backend APIs
- 💻 React frontend dashboard
- 📈 Real-time stock visualization

The system fetches stock data, processes it using ML models, and predicts future stock values.

---

# ▶️ How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

---

## 2️⃣ Run Machine Learning Server

```bash
cd stock-ml-project
python3 app.py
```

---

## 3️⃣ Run Spring Boot Backend

```bash
cd stockApp
mvn spring-boot:run
```

Backend will run on:

```bash
http://localhost:8080
```

---

## 4️⃣ Run Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```bash
http://localhost:5173
```
